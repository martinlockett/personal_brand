**Badge** — a small, restrained status or category marker. Use for tags, statuses, counts.

```jsx
<Badge>Essay</Badge>
<Badge tone="navy">New</Badge>
<Badge tone="brass">Featured</Badge>
<Badge tone="teal">Draft</Badge>
```

Tones: `neutral` (paper, default), `navy`, `brass` (reserve for the one featured thing), `teal`. Pill-shaped, 13px medium. Keep labels short.

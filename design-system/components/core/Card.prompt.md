**Card** — a raised paper surface for grouping content; warm soft shadow, hairline border, restrained radius.

```jsx
<Card brassEdge>
  <Eyebrow>Essay</Eyebrow>
  <h3>On thinking before speaking</h3>
  <p>…</p>
</Card>
```

`elevation`: `flat` · `raised` (default) · `floating`. Set `brassEdge` to add the signature 2px brass hairline along the top — reserve for a featured card. Override `padding` as needed.

**Button** — the brand's primary action control; quiet, classical, confident. Use it for any clickable action.

```jsx
<Button variant="primary">Read the guide</Button>
<Button variant="accent">Get in touch</Button>
<Button variant="secondary" size="sm">Download CV</Button>
```

Variants: `primary` (Ink Navy fill — default), `accent` (Brass — reserve for the one action that matters most on a view), `secondary` (paper outline), `ghost` (text-only, Slate Teal). Sizes: `sm` · `md` · `lg`. Supports `iconLeft` / `iconRight`, `disabled`. Hover darkens the fill; press nudges down 0.5px; focus shows a brass ring. No bounce — motion stays calm.

**Wordmark / Monogram** — the brand's logo system. Always give the mark generous clear space; whitespace is the brand.

```jsx
<Wordmark />                         {/* monogram + name */}
<Wordmark motto size="lg" />         {/* full formal lockup */}
<Wordmark showMonogram={false} />    {/* name only */}
<Monogram size={40} />               {/* compact: avatar, favicon */}
<Monogram size={40} mono="ink" />    {/* monochrome */}
```

The `Monogram` is the ML ligature in an Ink Navy rounded square with a brass keyline. The `Wordmark` sets "Martin Lockett" in Fraunces Semibold. Never stretch, outline, add shadow, or place on a busy background. Monochrome must work first — colour is the last layer.
